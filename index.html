✏️ Máy vẽ 2D

Mô tả: Máy vẽ 2D vẽ theo từng nét, có cơ cấu nâng hạ bút.

Cấu trúc: Trục X, Y điều khiển mặt phẳng, trục Z dùng servo nâng hạ bút.

Công nghệ:

  • Firmware: GRBL
  • PlatformIO
  • LaserGRBL / LightBurn

Ngôn ngữ: C

Quy trình: Thiết kế → G-code → Vẽ

Kết quả: Vẽ được hình cơ bản và PCB đơn giản

🎬 Demo 💻 GitHub

🖨️ Máy in 3D

Mô tả: Máy in 3D công nghệ FDM in từng lớp nhựa.

Cấu trúc: Trục X, Y đơn và trục Z đôi, khung in 3D.

Công nghệ:

  • Firmware: Marlin
  • PlatformIO
  • Ultimaker Cura

Ngôn ngữ: C

Quy trình: Thiết kế → Cura → G-code → In

Kết quả: In chi tiết nhỏ phục vụ dự án

🎬 Demo 💻 GitHub
