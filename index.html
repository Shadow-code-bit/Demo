✏️ Máy vẽ 2D COREXY

Mô tả: Máy vẽ 2D điều khiển bằng Arduino, có cơ cấu nâng hạ bút.

Cấu trúc: Trục X, Y chạy song song điều khiển mặt phẳng bằng một dây đai duy nhát, trục Z dùng servo.

Công nghệ:

  • GRBL (nhưng phải bật chế độ crexy)
  • PlatformIO / Arduino ide
  • LaserGRBL / LightBurn

Ngôn ngữ: C

Kết quả: Vẽ hình cơ bản và PCB đơn giản, chi tiết chuẩn xác ít lệch hơn so với máy kia
